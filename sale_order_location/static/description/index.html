Overview

In this module, user will be able to add the location of perticular product. When user will select the product in sale order, the locations will be filtered according to the selected product. And an additional button is added named "Confirm Stock and Proceed" which will check that the location is selected or not. Also the location field will show the name of the location and available stock at perticular location. When the sale order gets confirmed and delivery is created, the quantity of products will be updated with selected locations. And the location is also added to quotation report. This module will help user to select the location for the product sale.
